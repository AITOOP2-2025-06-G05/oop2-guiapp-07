# oop2-guiapp-07


実行方法

1. Gzou/main.pyを実行する
2. GUIのウィンドウが開く
3. キャプチャーボタンを押す
4. キャプチャした画像が画面に表示される
5. 白黒変換ボタンを押す
6. 白黒変換された画像が表示される


担当者

GUI作成者：塚田

画像処理アプリルール

実行したらGUIのウィンドウが表示されてその後、２種類のボタン(写真とる、白黒変換)を押すとそれに応じた画像が出力される。


GUIアプリケーションとしてPySide6のウィンドウが最初に表示されること

ボタンなどユーザーによるGUI操作で写真取得，画像合成の処理ができること
・写真を撮るボタンの追加
・白黒変換ボタン

処理状態（取得した画像や合成後の画像の表示など）がわかること
・撮った写真をそのまま出力する

・白黒画像にして、できた画像を出力する
