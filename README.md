# Gzou

- 写真を撮ってそれを白黒変換するためのGUIアプリ

## 実行方法

1. Gzou/window.pyを実行する
2. GUIのウィンドウが開く
3. キャプチャーボタンを押す
4. キャプチャした画像が画面に表示される
5. 白黒変換ボタンを押す
6. 白黒変換された画像が表示される
